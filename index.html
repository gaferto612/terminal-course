What you'll learn

🐧

Bash & Unix tools

The shell that powers Linux servers, macOS, and WSL. Navigation, pipes, grep, find, and the philosophy behind it all.

🪟

PowerShell

Microsoft's object-oriented shell. Cmdlets, the pipeline that passes data (not text), and why it's different.

⚡

Zsh, Fish & friends

The modern shells. Why macOS switched to Zsh, what Fish does differently, and how to pick one.

📜

Scripting

From running commands to writing programs. Variables, loops, conditions, functions — in Bash and PowerShell.

🔧

The wider toolbox

Git from the CLI, SSH, package managers, curl, jq, Python virtualenvs, tmux, and dotfiles.

🧪

Hands-on exercises

Every module ends with practice problems. Solutions included. Type the commands yourself — that's the whole point.
